What It Does

Opportunity Confidence evaluates a sales opportunity across four equally weighted dimensions, then turns the analysis into an easy-to-share confidence report with an overall score, recommendation, rationale, and expandable supporting detail.

Commercial Fit Assesses pain, value, buyer access, budget signal, and procurement path.
Technical Fit Checks whether Shopify capabilities, integrations, and data requirements have a credible path.
Momentum Reviews buyer engagement, next steps, urgency, and whether progress is buyer-owned.
Launch Confidence Evaluates time to value, implementation ownership, dependencies, launch plan, and first-value readiness.

Why Teams Should Use It

The skill helps teams make qualification decisions with less ambiguity and stronger evidence discipline.

  • Identifies whether an opportunity is worth continued sales and solutioning investment.
  • Surfaces commercial, technical, buyer, and launch risks before they become blockers.
  • Highlights missing information when the score cannot be trusted yet.
  • Separates real buyer momentum from internal optimism.
  • Creates a professional HTML report for AEs, SEs, leadership, and launch teams.
  • Supports consistent decisions across Proceed, Hold, Need Information, and Potential DQ outcomes.

Install From GitHub

Use the command for your tool. The repository URL is already filled in, so testers can paste the command without customization.

Prefer a download? Use the packaged zip: opportunity-confidence.zip. Extract it, then move the extracted opportunity-confidence folder into the matching skill folder for Claude, Cursor, Codex, or Pi.

Zip Download No git

Download:
https://github.com/TerryKealey/opportunity-confidence/raw/main/opportunity-confidence.zip

Extract, then move the opportunity-confidence folder into:
$HOME/.claude/skills/
$HOME/.cursor/skills/
$HOME/.codex/skills/
$HOME/.pi/skills/

Claude Skill folder

git clone https://github.com/TerryKealey/opportunity-confidence.git \
  "$HOME/.claude/skills/opportunity-confidence"

Cursor Skill folder

git clone https://github.com/TerryKealey/opportunity-confidence.git \
  "$HOME/.cursor/skills/opportunity-confidence"

Codex Skill folder

git clone https://github.com/TerryKealey/opportunity-confidence.git \
  "$HOME/.codex/skills/opportunity-confidence"

Pi Skill folder

git clone https://github.com/TerryKealey/opportunity-confidence.git \
  "$HOME/.pi/skills/opportunity-confidence"

# If your Pi setup reads from the agent skill path:
ln -s "$HOME/.pi/skills/opportunity-confidence" \
  "$HOME/.pi/agent/skills/opportunity-confidence"

After installing, restart the agent or start a new chat so the skill loader can discover the new SKILL.md file.
