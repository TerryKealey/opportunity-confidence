# Opportunity Confidence

Opportunity Confidence is a Shopify sales qualification skill for reviewing B2B and Plus opportunities. It scores opportunities across Commercial Fit, Technical Fit, Momentum, and Launch Confidence, then produces a clean HTML confidence report with one of four recommendations: Proceed, Hold, Need Information, or Potential DQ.

## Install

Clone this repository into the skill folder for your agent:

```bash
git clone https://github.com/TerryKealey/opportunity-confidence.git "$HOME/.claude/skills/opportunity-confidence"
```

Use the GitHub Pages site for platform-specific install commands:

```text
https://terrykealey.github.io/opportunity-confidence/
```

## Test Prompt

```text
Run Opportunity Confidence for St. Croix Rods.
```

Replace `St. Croix Rods` with the opportunity or merchant name you want to evaluate.
